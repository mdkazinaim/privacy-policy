Privacy Policy This Chrome extension does not collect, store, or share any personal data. All functionality is performed locally within the user's browser. The extension only accesses the active tab when the user explicitly interacts with it to inspect fonts, colors, and UI elements. No browsing data is transmitted to external servers. Permissions: - activeTab: Used to access the current page for inspection. - scripting: Used to extract styling information. - storage: Used to save user preferences locally.
